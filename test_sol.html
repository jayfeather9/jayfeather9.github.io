题解

I 组三角形

问题分析

若总计有偶数条木棒，先手显然可以保证长度为 ~ 的木棒被取完，这时显然可以拼成三角形。

若总计有奇数条木棒，称长度为到的木棒为短木棒，其余为长木棒。先手取长木棒，后手就拿短木棒中最长的；先手取短木棒，后手就拿长木棒中最短的。这样可以保证无法拼成三角形。

于是我们只要判断 为奇数还是偶数，再输出即可。

参考代码

#include <stdio.h>
int main()
{
    int n, t;
    scanf("%d", &t);
    while (t--) //重复t次读入数据并处理
    {
        scanf("%d", &n);
        if (n % 2 == 1)//判断奇偶，奇数则输出No，偶数则输出Yes
            printf("No\n");
        else
            printf("Yes\n");
    }
    return 0;
}
